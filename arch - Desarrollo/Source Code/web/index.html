About Us

arch es la unión de distintos proyectos Open Source que ofrecían servicios para Arquitectos o estudiantes de Arquitectura. La filosofía de arch es el trabajo colaborativo: Si necesitas ayuda, la encontrarás en arch. La mayoría de servicios ofrecidos en arch se desprenden de distintos feedbacks ofrecidos por los usuarios. No dudes en contactarte vía el correo de los desarrolladores para ofrecer tus ideas para que arch siga creciendo.
